# Backlog de Jeux Vidéo - API REST

Projet réalisé dans le cadre de la SAE **Développement & Déploiement d'une Application Web RESTful Conteneurisée** (Sup-Galilée).

# Contexte

Cette application est une **API REST** permettant à des utilisateurs de gérer leur **backlog personnel de jeux vidéo** : ajouter des jeux, suivre leur progression (à jouer / en cours / fini / abandonné), noter les jeux et écrire des avis.

# Stack technique

- **Backend** : Python + Flask
- **ORM** : SQLAlchemy
- **Base de données** : PostgreSQL
- **Conteneurisation** : Docker + Docker Compose
